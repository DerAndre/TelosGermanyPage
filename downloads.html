Downloads

Please feel free to verify downloads with our GPG Key

To ensure you got the right files, check back the SHA512SUMS and SHA512SUMS.sig

To verify you need gpg installed and enter the following commands:
gpg --import telosGermanyPub.key (if not already imported)
gpg --verify SHA512SUMS.sig SHA512SUMS

AppArmor Profile

We created an AppArmor Profile for Sqrl to minimize the risk caused by software vulnerabilities. AppArmor allows you control of access and processes and protects your system and application against zero-day attacks. Just Download the profile and installation guide here.

TelosPortal

TelosPortal is a wallet tool to handle your EOS and TLOS and is forked from the EOSToolKit. It is still under development but will be published here soon.

Scatter

Scatter is a single-sign-on-tool for blockchain applications and is compulsory in order to use the TelosPortal. You can get the latest release from GitHub.

EOSToolKit

EOSToolKit is a community driven Toolkit for the EOS network. You can either directly use the ToolKit here or get the code from GitHub.

Sqrl - Telos Foundation

At the Telos Foundation Website, you can download Sqrl which is a wallet designed for the Telos blockchain. It enables producer voting and common wallet commands. Additionally, the Telos White Paper and public media guidelines are downloadable, there.
